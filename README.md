Predicting Local Protein 3D Structures Using Clustering Deep RNN 

# Description

This application predicts local 3D protein structures from amino acid sequences using clustering techniques and Deep Recurrent Neural Networks (RNN). It analyzes sequence patterns, groups similar structural fragments, and learns sequential dependencies to improve prediction accuracy for bioinformatics research and drug discovery.

 Technologies Used

Python

NumPy & Pandas

Scikit-learn (Clustering)

TensorFlow / PyTorch (Deep RNN – LSTM/GRU)

Matplotlib

 Features

Amino acid sequence encoding

Structural fragment clustering

Deep RNN-based prediction

Performance evaluation metrics

Visualization of training results
